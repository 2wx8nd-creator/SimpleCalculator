#SimpleCalculator

[주요 기능]
1. 사칙연산
2. C, CE, del 기능
3. 자동 포커스
4. 입력이 없을 시 0 출력
5. 키보드 입력 가능 (숫자, 연산기호, dot, Enter, Backspace)
6. +/- 버튼 : 마지막 엔트리 숫자의 부호 변경
